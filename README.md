# cadastro-clientes

Sistema web simples de cadastro de clientes desenvolvido com HTML, CSS e JavaScript puro.

Sobre o projeto

Este projeto foi desenvolvido com os conhecimentos adquiridos em desenvolvimento web até o momento.
O sistema permite cadastrar clientes e visualizar a lista em tempo real.
Conforme avanço nos estudos, pretendo evoluir o projeto com:

- Integração com banco de dados SQL
- Back-end com Java e Spring Boot
- Interface com Angular

Funcionalidades

- Cadastro de clientes com nome, CPF, e-mail e telefone
- Validação completa dos campos via JavaScript
- Destaque visual nos campos com erro
- Capitalização automática do nome ao cadastrar
- Listagem dos clientes cadastrados

Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

Como executar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador

Aprendizados

- Manipulação do DOM
- Validação de formulários com JavaScript
- Eventos de formulário com preventDefault
- Criação dinâmica de elementos HTML
- Uso de expressões regulares (Regex)
